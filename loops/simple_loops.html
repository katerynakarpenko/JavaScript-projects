#2 task

Write a program in which using the console.log output all numbers from 1 to 100 with two exceptions. For numbers that are multiples of 3, instead of a number "Fizz" is displayed, and for numbers that are multiples of 5 (but not 3), "Buzz".
When it works, modify the program so that it prints "FizzBuzz" for numbers that are divisible by both 3 and 5 (and still type "Fizz" or "Buzz" for numbers that are multiples of only one of them).

Result

See in Console, Press F12

#3 task

Write a program that creates a string representing an 8 x 8 grid, using newline characters to separate lines. In every position the grid is either a space or the "#" symbol. These symbols must be located in checkerboard order. Passing this string to the console. log, you should get something like this:
 # # # #
# # # #
 # # # #
# # # #
 # # # #
# # # #
 # # # #
# # # #
If you have already written a program that generates this pattern, define it binding size = 8 and modify the program so that it works for any size, displaying a grid of the specified width and height.

Result

See in Console, Press F12
