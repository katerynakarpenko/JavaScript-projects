#2 task

Recursion. One way to determine is a positive integer even or odd:
  • zero is even;
  • 1 is odd;
  • the parity of any other number N coincides with the parity of N - 2.
Define a recursive function isEven corresponding to this description. The function must accept one parameter (a positive integer number) and return a logical value. Check this function on numbers 50 and 75. See how it behaves for -1. Why? Can you think of a way to fix this?

Result

See in Console, Press F12

#3 task

Counting letters.Write a countBs function that accepts a string as the only one argument and returns a number indicating how many large letters ~B~ is contained in this line.
Then write a countChar function that behaves like countBs, except that takes the second argument, indicating which ones symbols need to be counted (instead of counting only large ones letters ~B~). Rewrite countBs to use this new function.

Result

See in Console, Press F12
